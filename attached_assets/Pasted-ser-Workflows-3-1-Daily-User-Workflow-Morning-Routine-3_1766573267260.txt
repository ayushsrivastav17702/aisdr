Find new prospects using AI prospecting
Enrich prospect data
Build targeted lists
Add to campaigns

Send scheduled manual follow-ups
Complete promised actions

Respond to any urgent replies

Performance Review:

Check daily quota achievement

3.2 Campaign Creation Workflow
Step 1: Planning (5 minutes)

Define campaign objective (meetings, replies, awareness)
Choose target audience (industry, role, company size)
Select campaign type (cold, warm, re-engagement)

Step 2: Prospect List (10 minutes)

Use AI prospecting to find prospects OR
Upload CSV file
Review and refine list
Set expected volume

Step 3: Sequence Design (10 minutes)

Choose number of touchpoints (recommended: 5-7)
Select channels (email, LinkedIn, SMS)
Set timing between touches
Use template or start from scratch

Step 4: Message Creation (20-30 minutes)

Input value proposition and key points
AI generates message variants
Review and edit messages
Customize personalization
Add call-to-actions
Create subject lines

Step 5: Configuration (5 minutes)

Set daily send limits
Choose mailboxes
Configure A/B tests
Set up tracking

Step 6: Review & Submit (5 minutes)

Preview full campaign
Check for errors
Submit for manager approval OR
Schedule to launch

Step 7: Monitor (Ongoing)

Check first 24 hours closely
Respond to early replies
Adjust if needed

3.3 Reply Handling Workflow
When Reply Arrives:

Notification Received:

Push notification or email alert
Check reply sentiment classification


Read & Assess (2 minutes):

Read full conversation thread
Review prospect profile
Check engagement history
Determine response needed


Respond (5-10 minutes):
If Positive Reply:

Thank them for responding
Gauge interest level
Suggest meeting time
Include calendar link
Add to qualified pipeline

If Neutral Reply (Question):

Answer question thoroughly
Provide relevant resources
Keep conversation going
Move toward meeting

If Negative Reply:

Thank them politely
Ask permission to follow up later
Respect their decision
Archive and move on

If Out of Office:

Snooze for return date
Set reminder to follow up


Update Systems (2 minutes):

Add notes in AiSDR
Update prospect status
Set next action



3.4 Meeting Booking Workflow
When Prospect Shows Interest:

Confirm Interest:

"Would Tuesday at 2pm work for a 30-minute call?"
Provide 2-3 time options
Include calendar link


Send Meeting Invite:

Use integrated calendar
Include video call link
Add agenda or prep materials
Send confirmation email


Meeting Preparation (Day Before):

Research prospect and company
Review conversation history
Prepare questions
Check qualification criteria


Meeting Reminder (Day Of):

System sends auto-reminder
Verify prospect confirmed
Join meeting on time


Post-Meeting (Within 1 hour):

Log meeting notes in AiSDR
Update opportunity stage
Send follow-up email
Schedule next steps
Hand off to AE if qualified

5. User Success Metrics & KPIs
Activity Metrics

Campaigns launched per month: Target varies (typically 3-5)
Prospects contacted per day: Target 50-100
Emails sent per day: Target based on campaign volume
Reply response time: Target <2 hours during work hours
Active campaigns: Maintain 3-5 simultaneously

Performance Metrics

Meetings booked per month: Target based on company quota
Reply rate: Target 5-8.5%
Positive reply rate: Target >50% of replies
Open rate: Target 65-70%
Meeting show rate: Target >75%
SQL conversion rate: Track and improve

Quality Metrics

Message quality score: Target >4/5 (from manager reviews)
CRM data accuracy: Target 100%
Compliance adherence: Target 100% (no violations)
Campaign approval first-try rate: Target >80%

Efficiency Metrics

Time to create campaign: Target <1 hour
Time to respond to reply: Target <2 hours
Campaigns completed vs. started: Target >80%
Prospect contact rate: Target >90% of list

7. User Best Practices
Campaign Creation Best Practices

Research First:

Understand target audience deeply
Research common pain points
Know your value proposition
Study successful campaigns


Message Quality:

Keep emails 50-125 words
Clear value proposition
Strong call-to-action
Personalize beyond {{FirstName}}
Test subject lines


Sequence Strategy:

5-7 touchpoints optimal
Mix channels (email + LinkedIn)
Vary your approach each touch
Respect timing (2-3 days between)
Always provide value


Testing & Learning:

Run A/B tests consistently
Only test one variable at a time
Give tests enough volume (minimum 100 sends)
Document learnings
Apply learnings to future campaigns



Reply Management Best Practices

Response Speed:

Respond within 2 hours if possible
Set up push notifications
Check inbox 3-4 times per day minimum
Prioritize hot leads first


Reply Quality:

Always add value in response
Answer questions completely
Be conversational, not salesy
Match their tone
Keep responses concise


Meeting Booking:

Suggest specific times
Make scheduling easy (calendar link)
Confirm 24 hours before
Send reminder 1 hour before
Follow up no-shows promptly



Productivity Best Practices

Time Blocking:

Inbox time: Morning, midday, end of day
Campaign creation: Deep focus blocks
Prospecting: Daily consistent time
Learning: Friday afternoons


Prioritization:

Hot leads first
Quick wins next
Long-term projects last
Don't neglect follow-ups


Organization:

Use consistent naming (campaigns, lists)
Tag everything appropriately
Keep notes detailed and current
Archive completed items
Clean up weekly




8. User Troubleshooting Guide
Common User Issues
Issue: Low Reply Rates
Possible Causes:

Poor targeting (wrong ICP)
Generic messaging
Weak value proposition
Bad timing
Deliverability issues

Solutions:

Refine ICP and target list
Increase personalization
Clarify value proposition
Test different send times
Check email health scores
Get message feedback from manager

Issue: Campaigns Stuck in Approval
Possible Causes:

Manager backlog
Quality issues
Incomplete information
Compliance concerns

Solutions:

Follow approval checklist
Run self-review before submission
Add context notes for manager
Follow up with manager if >24 hours
Review previously approved campaigns

Issue: Low Open Rates
Possible Causes:

Bad subject lines
Email going to spam
Wrong send times
List quality issues

Solutions:

Test different subject lines
Check spam score before sending
Experiment with send times
Verify email addresses
Check domain health

Issue: Can't Find Prospects
Possible Causes:

ICP too narrow
Limited data access
Wrong search criteria

Solutions:

Broaden ICP criteria
Try different keywords
Use AI prospecting suggestions
Check data provider credits
Request additional data access

Issue: Meetings Not Booking
Possible Causes:

Not asking directly enough
Complicated scheduling process
Poor qualification
Follow-up gaps

Solutions:

Include calendar link in every reply
Offer specific time options
Qualify interest better
Set follow-up reminders
Reduce friction in booking
