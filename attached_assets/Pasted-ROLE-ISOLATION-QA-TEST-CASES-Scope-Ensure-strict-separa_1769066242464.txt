ROLE ISOLATION – QA TEST CASES
Scope

Ensure strict separation between:

Super Admin

Manager

User (SDR)

Across:

Authentication

Authorization

Routing

API access

Session handling

1️⃣ Authentication & Login
TC-AUTH-01: Super Admin login redirects correctly

Steps

Login as Super Admin
Expected

Redirects to /super-admin/dashboard

JWT role = super_admin

TC-AUTH-02: Manager login redirects to Manager dashboard

Steps

Login as Manager
Expected

Redirects to /manager/dashboard

Manager UI only

JWT role = manager

TC-AUTH-03: User login redirects to User dashboard

Steps

Login as User (SDR)
Expected

Redirects to /user/dashboard

User UI only

JWT role = user

TC-AUTH-04: No role fallback allowed

Steps

Login as User

Modify frontend role to manager
Expected

Backend denies access

403 Forbidden
