Total load becomes unbounded

Fix (Backend)

Active campaigns per tenant

2️⃣ SDR Explosion Risk

Risk

Manager adds too many users

Each SDR sends small volumes → total abuse

Fix

Enforce:

Max users per plan

Per-user quota auto-derived from tenant cap

3️⃣ Manager Can Starve Queue

Risk

One manager floods email queue

Other managers/SDRs starved

Fix

Queue partitioning:

Tenant → Manager → User

Weighted fair scheduling

4️⃣ No Manager-Level Kill Switch

Risk

One rogue manager forces Super Admin intervention

Fix

Manager-level pause:

Pause all campaigns under that manager

Does NOT pause tenant

5️⃣ No Blast Radius Isolation

Risk

Bad data upload (CSV / API) breaks processing

Fix

Validate at manager boundary:

Prospect count limits

Dedup before enqueue
